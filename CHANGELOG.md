# Changelog

All notable changes to Pi Voice will be documented here.

This project follows a simple changelog format inspired by Keep a Changelog and uses semantic versioning after the first public release.

## [0.1.0] - Unreleased

### Added

- Initial Pi package structure.
- `/voice` command for persistent hold-Space dictation mode.
- Local `whisper-cli` transcription backend by default.
- Explicit OpenAI-compatible transcription backend.
- `ffmpeg` microphone capture.
- Animated spectrum widget below the prompt editor.
- Privacy and endpoint-safety guardrails.
- OSS documentation files and issue templates.
