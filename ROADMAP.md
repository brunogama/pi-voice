# Roadmap

This roadmap is intentionally small and may change based on user feedback.

## Near term

- Improve live voice visualizer fidelity and terminal compatibility.
- Add more robust manual tests for hold-Space behavior across terminals.
- Add screenshots or a short preview GIF for the Pi package gallery.
- Improve first-run setup guidance for `ffmpeg`, `whisper-cpp`, and model files.

## Later

- Optional configurable keyboard shortcut in addition to Space.
- Optional transcript post-processing before insertion.
- Optional support for more local transcription servers.
- Better status display for long transcriptions.

## Non-goals

- No always-on background microphone listener.
- No silent cloud upload.
- No automatic prompt submission.
- No bundled Whisper model downloads during package install.
